# 26-IGRUS-플젝대회 web 부문 제출

1. 실행 방법

- step 1. 코드를 실행하면 '영수증'이라는 제목과 날짜가있는 칸, 품목칸, 가격칸, 추가 칸이 나오면 성공
- step 2. 날짜가 있는 칸에 원하는 날짜를 선택! >> 영수증 아래에 있는 날짜가 선택한 날짜로 바뀝니다.
- step 3. 품목에 소비한 품목 이름을 작성하고 가격을 입력한 뒤 추가를 누르기!

  3-1. 영수증 화면에 품목과 가격이 반영되면 성공

  3-2. 0원, 음수 모두 입력 가능
  
- step 4. 입력을 끝내고 이미지를 저장하고싶다면 이미지 저장 버튼을 누른 뒤 파일 명, 저장 장소를 선택하면 다꾸용 영수증 생성 완료!
- step 5. 워드 또는 한글 파일로 원하는 크기로 조정한 후 출력하여 사용

   5-1. 취향에 맞게 도장, 스티커, 그림을 추가하면 더 귀여워짐

   5-2. 종이를 구겼다 펴서 붙이면 멋진 질감 완성!

2. 작품소개


다꾸를 할 때 영수증을 붙이게 됩니다.

'나 이만큼 소비했어요!'

'나 오늘 이거삼 ㅎ'

'이 카페 메뉴 맛남' 등등의 이유로 영수증을 붙이는데

가장 큰 단점 바로 >>시간이 지나가면 백지가 됨<< 이라는 현상이 나타나게 되죠...

그렇기 때문에 따로 영수증 메모지가 존재하기도합니다.

하지만 그런 메모지는? >>내가 원하는길이대로 조절을 못 한다!<<

였습니다.

그래서 고안한 방법!

원하는 길이만큼 조정되는 레이아웃 어디없나...?

그리하여 gpt를 갈궈 탄생한 다꾸 영수증!

또한 다꾸의 매력은 바로 '''자유도''' 인데요

이런 자유도를 살려주기 위해

- 품목에 공백 입력

- 가격에 0원, 음수 입력

이 모두 가능하도록 만들었습니다.


이상입니다.
